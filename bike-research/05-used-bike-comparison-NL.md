## New vs Used — The Trade-Off

| Factor | New Cube Attain C:62 SLX (€2,499) | Used TT/Tri Bike (€2,000–€3,000) |
|---|---|---|
| **Groupset** | 105 Di2 (2025, electronic) | Often Ultegra Di2 (older, 11-speed) |
| **Frame** | Modern C:62 carbon, warranty | Older carbon, no warranty |
| **Wear** | Zero | Cassette/chain/brakes may need replacement |
| **Tyre clearance** | 34mm | Usually 25-28mm |
| **Gearing** | 50/34 + 11-34 (climbing friendly) | Often 52/36 or 53/39 + 11-25/28 (racing gearing) |
| **Brakes** | Hydraulic disc | Often rim brakes (older models) |
| **Aero** | No tri-bars (add-on) | Built-in aero cockpit |
| **Fit** | Road geometry + clip-ons | Dedicated tri geometry (steep seat tube) |
| **Warranty** | 6 years (frame) | None/limited |
| **Resale value** | Depreciates immediately | Slower depreciation |

## Hidden Costs of Buying Used

| Item | Cost (€) | Notes |
|---|---|---|
| Bike fit | 150-250 | Essential for comfort/performance |
| New tyres | 80-150 | Old tyres are risky |
| New chain + cassette | 100-200 | If drivetrain is worn |
| Brake pads | 30-60 | If worn |
| Clip-on aero bars | 50-200 | Only if not included |
| Full service | 100-200 | Peace of mind |
| **Total potential extra** | **€510-1,060** | |

A used bike at €2,000 may cost €2,600+ after making it race-ready.

1. **Used Cube Attain C:62** (~€1,200-1,800) — same frame you're considering, proven for this course, add clip-ons + get a bike fit
2. **Used Canyon Endurace CF 7 Di2** (~€2,200-2,600) — lighter than Cube (8.2 kg), Di2, more tyre clearance (35mm), excellent climber
3. **Canyon Speedmax CF 7 Disc** (~€2,500) — if you want a true tri bike, modern disc brakes, but harder gearing for the climbs
4. **Giant Defy Advanced 2 used** (~€2,000-2,400) — 40mm clearance, very comfortable, good climber

**Avoid for Alpe d'Huez:**
- Old TT bikes with rim brakes and 53/39 + 11-25 gearing (you'll struggle on climbs)
- Aluminium TT bikes (Cube Aerium HPA) — heavy and uncomfortable
- Very old carbon frames (pre-2015) — unknown fatigue, no warranty
